# Papa Test CSV Sensory Dashboard

A lightweight, single-page dashboard for exploring sensory CSV data with filters,
charts, and a pivot table. Built with plain HTML/JS, Papa Parse, and Chart.js.

## How to run

1. Open `index.html` in a browser.
2. Upload a CSV file (export from Excel if needed).
3. Use the filters to explore results.

## Notes

- This is a static, client-side app. No server required.
- If your CSV headers differ, update the `COL` map in `app.js`.
